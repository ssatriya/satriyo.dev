## Overview
Pointify is a specialized student behavior management platform that implements a point-based discipline system. It allows educators to track disciplinary incidents while simultaneously providing a reward system to incentivize positive student behavior.

## Tech Stack
- **Framework**: Laravel 11
- **Frontend**: Inertia.js, React
- **Language**: TypeScript, PHP
- **Styling**: Tailwind CSS, Shadcn UI
- **Database**: MySQL/PostgreSQL

## Key Features
- Discipline point tracking for monitoring student conduct.
- Reward point system to incentivize positive behavior.
- Real-time student progress and behavior dashboards.
- Seamless single-page application experience with Inertia.js.
- Responsive design for teachers and administrators on any device.
