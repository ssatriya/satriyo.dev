## Overview
Focuses on accessibility and user experience, providing a clean interface for reading and listening to the Quran.

## Tech Stack
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Data Fetching**: Axios

## Key Features
- Live search functionality across the Surah list.
- Audio playback integration for every Surah.
- Smooth loading states with skeleton screens for verses (Ayat).
- Mobile-friendly navigation.
