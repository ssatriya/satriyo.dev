## Overview
Lab PKM LUT is a specialized laboratory management platform built to streamline data entry, processing, and reporting workflows. It focuses on reliability and ease of use for lab personnel.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Database**: Drizzle ORM
- **Styling**: Tailwind CSS, Shadcn UI
- **Authentication**: NextAuth.js (based on typical Next.js setups)

## Key Features
- Dynamic data entry forms for lab results.
- Robust reporting system with export capabilities.
- Type-safe database interactions with Drizzle ORM.
- Modern, responsive dashboard interface.
