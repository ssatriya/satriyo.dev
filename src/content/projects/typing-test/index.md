## Overview
A minimalist and functional utility app designed for speed and accuracy assessment, inspired by Monkey Type.

## Tech Stack
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Key Features
- Real-time calculation of WPM (Words Per Minute) and Accuracy.
- Responsive interface focused on desktop usability.
- Instant reset functionality.
